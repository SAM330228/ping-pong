# ping-pong
Проект находится в репозитории
Игра реальзована с помощью Python и использованием библиотеки PyGame.
Игра создана для 2 игроков.
Первый игрок управляет клавишами w и s.
Второй игрок управляет стрелками мыши вверх и вниз.
Мяч отскакивает от ракетом, если он пролетит, то проиграет тот, кто пропустил мяч.
